# express-xcontroller
Node后端微服务框架，基于express-xcontroller中间件，自动路由映射控制器文件

[传送门：XServer官网文档](http://www.xserver.top)

框架目录结构
>
	├── app.js
	├── config
	│   ├── default.json
	│   ├── develop.json
	│   └── production.json
	├── node_modules
	├── package.json
	├── src
	│   └── controller
	└── xcontroller_modules
	    └── express-xcontroller
